About Me I am a developer with a keen interest in game development. I attended the University of York, in the United Kingdom, where I received a BSc Computer Systems degree.

Whilst studying, I also founded the university's game development society, DevSoc, which focused both on game development using the Unity game engine, as well as game theory.

I also work on my own open-source cross-platform 2D game engine, called the Dethsquare Engine, which leverages technologies like OpenGL and OpenAL (with planned Vulkan support).
